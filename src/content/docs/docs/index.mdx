## Apa Itu Gaman?

GamanJS adalah sebuah framework javascript yang befokus ke aplikasi backend. awalnya project ini di buat hanya sebagai hobi saja. dan selamat untuk saya ini adalah project terbesar yang pernah saya buat.

## Filosofi

Nama ***Gaman*** diambil dari kata-kata Jepang seperti "gaman shinasai" atau "gaman dekinai" yang berarti bersabar atau bertahan. Filosofi ini mencerminkan semangat untuk tetap sabar dan konsisten dalam mengembangkan sesuatu hingga tuntas.

## Motivasi

Berawal dari sekadar proyek hobi. Biasanya, proyek lain tak pernah selesai, tetapi entah bagaimana GamanJS berhasil mencapainya. Itulah sebabnya kini ada motivasi untuk terus mengembangkan kerangka kerja ini.

Dikembangkan oleh seorang individu dibawah nama 7TogkID, GamanJS terus berkembang dengan menambahkan berbagai fungsi dan utilitas yang berguna untuk pengembangan backend.

## Kenapa Bahasa Indonesia?

Karena sebenernya saya agak kurang bisa berbahasa inggris, dulu pernah membuat dokumentasi berbahasa inggris hasil translate + AI, bahasa jadi aneh dan kurang bisa di mengerti (kata orang orang yang makai gaman).
jadi saya putuskan untuk menargetkan pasar ke local dulu yaitu (Indonesia).

Dan saya berharap ada yang membantu kontribusi untuk pembuatan dokumentasi berbahasa Inggris :)

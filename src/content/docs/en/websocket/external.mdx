Websocket external is not built-in to GamanJS, what I mean here is that we just want to convey: *GamanJS can also install other Websocket libraries, for example like [socket.io](https://socket.io) and the like.*

## Socket.io

Here I will give an example of a websocket library that is widely known by developers in the world, namely [socket.io](https://socket.io).

### Install

import { PackageManagers } from "starlight-package-managers";

GamanJS is a framework for backend applications, you can install using your favorite package manager:

<PackageManagers

  pkg="socket.io"

  pkgManagers={["npm", "pnpm", "yarn", "bun"]}

  type="install"

 />

### Implementation

```diff lang="ts"

// index.ts

+ import { Server } from 'socket.io';

  defineBootstrap(async (app) => {

    const server = await app.mountSever(":3431");

+   const io = new Server(server);

+   io.on('connection', (socket) => {

+     console.log('a user connected');

+   });

  });

```

### Client Side

```diff lang="html"

+  <script src="/socket.io/socket.io.js"></script>

    <script>

+    const socket = io();

    </script>

```
